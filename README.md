# Glassmorphism Portfolio

Premium glassmorphism portfolio template built with React, Vite, and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

## Customize Content

Update data files in:

- `src/data/profile.js`
- `src/data/skills.js`
- `src/data/projects.js`
- `src/data/experience.js`
- `src/data/testimonials.js`

## Branding

- Update SEO tags in `index.html`.
- Replace icons and social preview in `public/`.
- Update colors in `tailwind.config.js` and `src/index.css`.

## Production

```bash
npm run build
```
